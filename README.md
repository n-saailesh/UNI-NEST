🏡UNI-NEST: Student Residence Finder
A platform that helps students find a suitable residence based on thier specfic preferences--built to empathize the significance of UX Journey/Service and Design.

✨Inspiration
This project was inspired off one of my own challenges. When my friend and I was near of completing my first year of university, we slowly realized that I needed to find a place to live in for my second year and moving forward. Frankly, we were searching all these different wesbite and estate rental 
platforms to find an arpartment and it was not easy. One problem that continously faced was that none of these known rental platforms was not student focused. Students are going have different needs comapred to other regular citizens in the city, for instance, the distance between home 
and school, the roommates, the noise levels, the rent amount, types of contracts and more. While we wre going this challeneg, we were also taking this course, UX Strategy and we decided that our entire project for this course was inspired off this challenege. After finding a few other stduents 
from different feilds of study, but with the same problem, we used this cahllenge to demossrtate a solution that also demosrated to students the importance of UX Journey/Service and Design. We used the Design Thinking Process (Empathize, Define, Brainstorm, Ideate, Test, Prototype). There 
were three deliverables and each deliverable was respsoible for a few steps of the UX Thinking Process. After receivng the insightful feedback for each deliverable, we created a high-fideity prototype of our platform UNI-NEST using Figma. 

With the permission of my colleagues, I created a website based off our high-fideity prototype. I used project-based learning to learn the web developing lanauages, **HTML, CSS and JavaScript**, used in this project. All in all, this was a great opportunity to showcase my skill. 
My goal of this website is to hopefully help students avoid same challenge I came across when finding my new home.


❓How does this promote the importance of UX Journey/Service and Design 
With the users’ feedback, we brainstormed ideas on how to better convey the idea of Journey/Service Design in UX. With the ideation, we decided that an application to help students find residence would better convey the importance of the concept. We created a low-fidelity prototype, 
however users’ found the chats’ questions to be repetitive at some tendency, thus we created an option using our Journey/Service Design knowledge in UX to lessen the repetition, by allowing users to create an account. It was implemented into the high-fidelity prototype and users 
found the feature to make the Journey process to be less tendency and more. This helps students understand how service design can be used to understand the struggles of an application, but also journey design can be used to better improve it.
